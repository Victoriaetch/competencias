Competencias

Una competencia integra conocimientos, habilidades, actitudes y valores para desempeñar funciones en un contexto determinado.

01

Operación de Sistemas Mecatrónicos

1.1 Implementar software y hardware específicos para el correcto desempeño de procesos industriales.

Se diseñaron e implementaron sistemas embebidos e industriales integrando hardware y software para el control de procesos físicos en tiempo real. En el área de microprocesamiento se utilizó el microcontrolador ATmega328P programado en lenguaje C a bajo nivel, configurando periféricos como ADC, PWM y UART. A partir de esto se desarrollaron sistemas de adquisición y control, como un sistema de gestión térmica con sensor LM35 y actuadores, incorporando comunicación serial para la recalibración dinámica del punto de operación sin modificar el código fuente. También se abordaron sistemas de control de motores, interfaces de usuario y sistemas de identificación RFID, integrando sensores, procesamiento y actuación en distintas aplicaciones.

En el área de sistemas de control se implementaron soluciones de automatización utilizando microcontroladores y plataformas industriales como OpenPLC bajo el estándar IEC 61131-3. Se desarrollaron sistemas de control por umbrales con histéresis mediante sensores ultrasónicos, módulos de relés y drivers de potencia para el accionamiento de motores y cargas eléctricas. El comportamiento de los sistemas fue modelado mediante diagramas GRAFCET, permitiendo validar la lógica secuencial y asegurar transiciones coherentes entre estados.

Además, los conocimientos relacionados con esta competencia se ven reflejados también en los laboratorios de electrónica analógica y digital, donde se desarrollaron circuitos basados en transistores, reguladores de voltaje, amplificadores operacionales y lógica combinacional y secuencial. Estos trabajos permiten complementar la capacidad de integración de hardware y software, al abordar desde el acondicionamiento de señales analógicas hasta el diseño de sistemas digitales de control, reforzando la capacidad de integración entre etapas analógicas, digitales y de control en sistemas embebidos.

03

Diseño de Sistemas Mecatrónicos

3.1 Fabricar equipos, sistemas y procesos Mecatrónicos de acuerdo con diseño

Diseño y fabricación de un sistema mecatrónico de adquisición y transmisión de datos ambientales, incluyendo el desarrollo de una PCB personalizada, ensamblado electrónico (SMD y THT) e integración en una carcasa impresa en 3D. El sistema fue implementado y validado experimentalmente, logrando un prototipo funcional.

3.2 Incorporar tecnologías (ya evaluadas) a procesos y sistemas Mecatrónicos

Integración de tecnologías existentes en un sistema embebido, incluyendo sensores ambientales, microcontrolador ATmega328PB y comunicación inalámbrica mediante Sigfox. Se implementó además la conexión con API externa para gestión y visualización de datos, asegurando interoperabilidad y eficiencia del sistema.

3.3 Generar insumos de proceso existentes para el diseño o rediseño de un sistema

Análisis y rediseño de un sistema de recolección de datos existente, identificando limitaciones en consumo, tamaño e integración. Se utilizaron herramientas de ingeniería como matrices multicriterio y evaluación de desempeño para seleccionar componentes y proponer mejoras, generando criterios técnicos para futuras optimizaciones.

04

Innovación e Investigación en Sistemas Mecatrónicos

4.1 Reconocer paradigmas tecnológicos tradicionales e innovadores presentes en una infraestructura o equipamiento industrial, con el propósito de optimizar sistemas y procesos mecatrónicos.

Análisis de paradigmas tecnológicos en sistemas de adquisición de datos, comparando una solución tradicional basada en Arduino con un sistema embebido optimizado. Se identificaron diferencias en consumo energético, tamaño, integración y capacidad de comunicación, incorporando tecnologías IoT como Sigfox para mejorar el desempeño del sistema.

4.2 Fabricar prototipos para sistemas y procesos mecatrónicos

Fabricación de prototipos mecatrónicos funcionales en el marco de proyectos integradores, incluyendo el desarrollo de un sistema de adquisición y transmisión de datos con PCB personalizada, así como un prototipo físico automatizado de difusión de fragancias. En ambos casos se realizaron tareas de ensamblado, integración de componentes y verificación de funcionamiento.

4.3 Colaborar en la integración de nuevas tecnologías a los sistemas mecatrónicos realizando pruebas / ensayos.

Integración de tecnologías IoT en un sistema embebido, incluyendo comunicación mediante Sigfox y sensores ambientales, realizando pruebas de funcionamiento, consumo energético y transmisión de datos para validar el desempeño y la confiabilidad del sistema desarrollado.
